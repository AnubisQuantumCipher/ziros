# zkf-ir-spec

`zkf-ir-spec` holds the formal IR specification and the verification-ledger
surfaces that define what ZirOS is allowed to claim. If a proof, backend, or
runtime boundary is not reflected here, it should not be described as shipped
truth elsewhere in the repository.

## Public API Surface

- Library crate: `zkf_ir_spec`
- Outputs: IR versioning, verification ledger, proof-boundary support metadata
