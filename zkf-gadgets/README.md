# zkf-gadgets

`zkf-gadgets` contains the reusable gadget registry for common proving
operations such as hashes, range checks, signatures, Merkle paths, and related
helper metadata. It is the typed catalog behind the gadget-facing surfaces in
the wider workspace.

## Public API Surface

- Library crate: `zkf_gadgets`
- Main exports: gadget registry/specification types and builtin gadget metadata
