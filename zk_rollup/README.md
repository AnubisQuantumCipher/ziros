# zk_rollup

`zk_rollup` is a workspace member for rollup-oriented proving experiments built
on top of ZirOS primitives. It is an application-layer crate rather than a
foundational proving component.

## Public API Surface

- Package: `zk_rollup`
- Scope: rollup-focused application logic on top of `zkf-core`, `zkf-backends`,
  and `zkf-dsl`
