# zkf-cli

`zkf-cli` is the operator-facing command-line interface for ZirOS. It exposes
audits, compile/prove/verify, app scaffolding, runtime planning, swarm and
telemetry controls, deployment helpers, and the demo pipeline.

## Public API Surface

- Binary: `zkf-cli`
- Main commands: `app`, `audit`, `compile`, `prove`, `verify`, `deploy`,
  `doctor`, `metal-doctor`, `runtime`, `swarm`, `telemetry`, `demo`
